# Space-Shooter-Game
A 2D arcade-style space shooter built with Python and Pygame. Control a spaceship, dodge enemy attacks, and destroy incoming enemies while trying to survive as long as possible.

## Features
* Smooth player movement & animation
* Shooting mechanics
* Enemy spawning, movement, & collision detection
* Health and score tracking
* Custom Pixel Art Graphics by Urav Singh
* Custom Parallax Starfield
* Dynamic Difficulty Scaling

## Built With
* Python
* Pygame
* Object Oriented Architecture
* Event Loop Handling
* State Management
* Delta Time/Frame Rate Independent movement

## What I Learned
This project helped me strengthen my Python programming skills while learning the fundamentals of game development. I gained experience with object-oriented programming, game loops, collision detection, event handling, and managing multiple game objects in real time. I also designed and created the pixel art assets used in the game.
